Bar Charts - Simple

  1. Schreiben Sie eine Funktion updateData, die ein sortiertes Array von Objekten mit den Properties cocktail und number erzeugt. Number sollte sich im Interval [0, 10] befinen mit einer erhöhten Wahrscheinlichkeit für 0 (zu Demonstrationszwecken).
  2. Definieren Sie Variablen für Höhe, Weite und Innenabstand Ihrer Graphik.
  3. Erzeugen Sie in #app ein svg-Element mit Höhe und Breite. Benutzen Sie d3.select.
  4. Erzeugen Sie ein Gruppenelement, das sie um den Innenabstand verschieben. Merken Sie sich das Gruppenelement in einer Variable. Alle späteren Manipulationen sollen bzgl. dieses Gruppenelements stattfinden.
  5. Erzeugen Sie Skalen für x und y. x ist eine Ordinale Skala (d3.scale.ordinal), während y eine lineare Skala ist (d3.scale.linear).
  6. Erzeugen Sie entsprechende Achsen mit d3.svg.axis und fügen Sie die jeweiligen Gruppenelemente hinzu. Sie sollten jetzt Achsen sehen können. Stylen Sie sie mit Hilfe von CSS.
  7. Erzeugen Sie ein Array von Daten und fügen Sie über die enter-Selektion Rechtecke in das Chart ein. Um die Attribute x, y, width, height zu berechnen, nutzen Sie die von Ihnen erzeugten Skalierungsfunktionen. Stellen Sie sicher, dass Sie das SVG-Koordinatensystem verstanden haben.
  8. Die Rechtecke sollen die CSS-Klasse .bar bekommen. Stylen Sie die Rechtecke.
