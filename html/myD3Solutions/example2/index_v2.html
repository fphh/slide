Bar Charts - Transitions

  1. Stylen sie das Pfad-Element der x-Achse weg (display: none)
  2. Definieren Sie eine Funktion update, die eine Zeile auf der Konsole ausgibt. Fügen Sie ein input-Element vom Typ button ein und setzen Sie onclick auf update. Sehen Sie den Output auf der Konsole, wenn Sie klicken?
  3. Implementieren Sie die Funktion update. Beim Klicken selektieren Sie alle Rechtecke über den CSS-Selektor .bar. Assozieren Sie neu erzeugte Daten zu dieser Selektion. Passen Sie die Attribute y, height an.
  4. Zum Animieren benutzen sie chained methods transition und duration.
