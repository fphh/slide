Aufgabe 1

  1. Erzeugen Sie ein Array mit 10 Elementen. Die Elemente sollen absteigend sortiert werden.
  2. Selektieren Sie das div mit der ID #app und fügen Sie eine SVG-Graphik ein. Speichern Sie die Selektion in einer Variable.
  3. Selektieren Sie alle Kreise innerhalb der selektierten SVG-Graphik und assozieren Sie das 10-elmentige Array. Dabei spielt es keine Rolle, dass die Kreise noch nicht existieren. Speichern Sie die Selektion in einer Variable.
  4. Mit der enter-Selektion fügen Sie jetzt die Kreise ein.
  5. Setzen Sie für jeden Kreis Attribute und Styles.
  6. Für die schönen Farben benützen Sie diese ordinale Skala. Als Domain setzen Sie den kleinsten und Größten Index Ihres Arrays.
